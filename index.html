Quantum entanglement is a phenomenon where two particles become linked so that measuring one instantly reveals the state of the other, no matter how much distance separates them. This connection means that observing a property of one particle immediately tells you what property its partner has, without any physical signal traveling through space between them. Despite this instant correlation across space, it cannot be used to send information faster than light because each individual measurement result remains random and unpredictable.

def is_prime(num):
    """Check if a number is prime."""
    if num < 2:
        return False
    for i in range(2, int(num ** 0.5) + 1):
        if num % i == 0:
            return False
    return True

Jargon Decoder

Terms Alex assumes you know. Now you do.

Lean Formalization
Translating a mathematical proof into code in the Lean programming language, where a computer verifies every logical step is correct. Like spell-check, but for math.
Fields Medal
The highest honor in mathematics, awarded every 4 years to mathematicians under 40. Often called "math's Nobel Prize" (math has no actual Nobel).
RL Compute
Computing power spent on Reinforcement Learning - where an AI learns by trial and error (like playing a game millions of times) rather than studying examples.
Tokens per Second
Speed measure for AI text generation. Each "token" is roughly ¾ of a word. 950 tok/s means generating ~700 words per second.
EUV Lithography
Extreme Ultraviolet Lithography - the most advanced chip-making technology. Uses ultra-short wavelength light to etch circuits at nanometer scale. Only ASML makes these machines (~$380M each).
GAA (Gate-All-Around)
Next-generation transistor design where the gate wraps entirely around the channel (instead of sitting on top). Better power efficiency and performance at tiny sizes like 2nm.
TOPS
Trillion Operations Per Second - a measure of AI processing power on a chip. 50 TOPS means 50 trillion math operations every second, dedicated to AI tasks.
NPU
Neural Processing Unit - a specialized chip built just for AI workloads. Separate from the CPU (general computing) and GPU (graphics). Optimized for the matrix math that AI models need.
CFTC
Commodity Futures Trading Commission - the US federal agency that regulates derivatives and futures markets. If they regulate it, it's officially a commodity (like oil, gold, or corn).
SHA-256
A cryptographic hash function. Takes any input and produces a fixed-length code that's practically impossible to reverse. Used in Bitcoin, HTTPS, and digital signatures.
Iron Dome
Israel's existing missile defense system. Uses physical interceptor missiles (~$50,000 each) to shoot down incoming rockets. Iron Beam (laser) is its potential successor at $4/shot.
PJM
The regional grid operator managing electricity transmission for 13 US states + DC, serving 65 million people. When PJM approves $11.8B in expansion, it's real contracted infrastructure.
AdderBoard
A competition to build the smallest possible AI model (fewest parameters) that can accurately add 10-digit numbers. Tracks how fast model efficiency is improving - from 121 parameters to 36 in one week.
Neural Engine
Apple's dedicated AI accelerator chip, built into every M-series Mac and recent iPhone. Extremely power-efficient (~80x more than Nvidia A100) but poorly documented for direct use.
